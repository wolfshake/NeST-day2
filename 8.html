BSc Data Science

BSc Data Science is a three-year undergraduate program that focuses on data analysis, statistics, programming, and machine learning. The course teaches students how to collect, analyze, and interpret large sets of data. Students develop skills to extract meaningful insights from data to help organizations make better decisions.

Key Subjects

  • Statistics
  • Python Programming
  • Machine Learning
  • Data Visualization

Career Opportunities

  • Data Analyst
  • Data Scientist
  • Machine Learning Engineer
  • Business Intelligence Analyst
